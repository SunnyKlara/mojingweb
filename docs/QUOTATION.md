## 二、市场定价分析

本项目属于**单品牌 DTC（Direct-to-Consumer）跨境电商独立站**，功能上覆盖了
品牌展示、在线交易、支付对接、订单履约、客服系统五个核心模块。
以下从三种主流建站方式进行对比：

### SaaS 平台方案（Shopify / BigCommerce）

使用 Shopify Basic 套餐（$39/月）+ 付费主题（$150–$350）+ 必要插件
（实时客服 $20–60/月、多语言 $20/月），首年综合成本约 **¥8,000 – 15,000**。

但存在两个长期成本：一是每笔交易 2.9% + $0.30 的支付手续费（年销售额 50 万时
手续费约 ¥15,000/年），二是平台月费和插件费用持续产生。
此外，Shopify 模板的定制上限明显，无法实现本项目中的自定义暗色主题、
实时 WebSocket 客服、完全自主的支付安全策略等功能。

### 外包公司定制开发

国内有跨境电商建站经验的外包公司（如杭州、深圳地区），同等功能范围
（自研架构 + PayPal 对接 + 双语 + 管理后台 + 客服系统 + 安全加固 + 部署），
报价通常在 **¥80,000 – 150,000**，开发周期 2–3 个月。
这个价格包含产品经理、UI 设计师、前端、后端、测试的团队配置。

### 独立开发者定制开发

在猪八戒、Upwork 等平台上，有 3 年以上全栈经验的独立开发者，
同等功能复杂度的项目报价在 **¥40,000 – 80,000**，开发周期 1.5–2 个月。
按行业通行的 ¥800–1,500/人天费率计算，本项目约 48 个有效工作日，
对应 **¥38,400 – 72,000**。

### 小结

| 方案       | 一次性成本        | 年持续成本         | 定制程度             |
| ---------- | ----------------- | ------------------ | -------------------- |
| Shopify    | ¥8,000 – 15,000   | ¥6,000 – 20,000+   | 低，受限于模板和插件 |
| 外包公司   | ¥80,000 – 150,000 | 维护费另计         | 高                   |
| 独立开发者 | ¥40,000 – 80,000  | 视维护约定         | 高                   |
| **本项目** | **¥25,000**       | **3 个月免费维护** | **高，完全自研**     |
